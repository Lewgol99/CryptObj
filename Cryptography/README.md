The CryptObj Protocol has the option for PKI by combining Asymmetric and Symmetric Ciphers using Key Wrapping. It is possible to use both Rivest–Shamir–Adleman (RSA) and Elliptic Curve Cryptography (ECC) with various keys and curves available. Additionally TLS 1.3 is also available to be deployed using the same curves.
